Features

📋

Real-time Usage

Monitor your 5-hour session and 7-day weekly limits at a glance.

🎨

Glassmorphism UI

Beautiful frosted glass design with smooth animations and hover effects.

🔒

Zero Token Cost

Uses OAuth usage API only. No Claude messages sent, no tokens consumed.

🔄

Auto Sync

Configurable auto-refresh: 5m, 10m, 30m, 1h, or manual.

🌐

Cross Platform

Native Swift on macOS, Node.js web widget on Windows & Linux.

🇬🇧

Bilingual

English and Korean language support built-in.
