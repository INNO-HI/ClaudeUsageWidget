Privacy Policy

Effective date: 2026-04-29 · Last updated: 2026-04-29

Claude Usage Widget ("the App") is an open-source desktop widget published by INNO-HI Inc. This policy explains what data the App accesses, where it goes, and what we do not do.

Data the App reads (locally only)

  • Claude Code OAuth credentials from ~/.claude/.credentials.json (created by the official Claude Code CLI). The App uses these to authenticate against Anthropic's usage API. They are never copied, transmitted to us, or persisted anywhere outside the file Claude Code already maintains.
  • Local config at ~/.claude-monitor-config.json — your sync interval, language preference, and toggle states (Launch at Login, notifications, etc.). Stored on your Mac only.

Network requests the App makes

  • GET https://api.anthropic.com/api/oauth/usage — fetches your usage percentages directly from Anthropic. No request body is sent. Zero token cost.
  • GET https://inno-hi.github.io/ClaudeUsageWidget/appcast.xml — Sparkle update check. Sends no personal data; only your standard HTTP user agent.

What the App does NOT do

  • It does not send messages to Claude or use any of your token quota.
  • It does not transmit your credentials, usage data, or any telemetry to INNO-HI or any third party.
  • It does not include analytics, ad SDKs, crash reporters, or fingerprinting libraries.
  • It does not request elevated privileges (no sudo, no system-level installation).

Auto-updates

The App uses Sparkle 2.x to check for and install updates. All update binaries are cryptographically verified with EdDSA signatures bound to a public key embedded in the App, and signed/notarized with our Apple Developer ID before distribution.

Open source

The full source code is available at github.com/INNO-HI/ClaudeUsageWidget under the MIT License. You can audit every line yourself.
